# Практика 2: Калибровка и эпиполярная геометрия

## Суть работы
Цель практики — калибровка камеры и анализ эпиполярной геометрии. Основные этапы:

1. **Сбор данных:** сделаны снимки шахматной доски для поиска внутренних углов (9x6).
2. **Калибровка камеры:** рассчитаны матрица камеры K и коэффициенты дисторсии, оценка RMS ошибки проекции.
3. **Коррекция дисторсии:** проверка изображений до и после применения `cv.undistort`.
4. **Сопоставление кадров сцены:** поиск ключевых точек с ORB, построение фундаментальной матрицы с RANSAC и визуализация эпиполярных линий.

## Импорт и пререквизиты
Использовались библиотеки:
- OpenCV
- NumPy
- Matplotlib
- Google Colab patches (для отображения изображений)

## Поиск углов шахматки и сбор данных
- Использовалась шахматная доска размером **9x6 внутренних углов**.
- Было сделано **17 снимков шахматки**.
- Углы успешно найдены на **X кадрах** (указать число).
- Для точной подстройки использовались критерии `cv.TERM_CRITERIA_EPS + cv.TERM_CRITERIA_MAX_ITER`.



## Выводы
- Калибровка выполнена успешно, RMS ошибка ~0.094 px, что указывает на точное соответствие точек.
- Дисторсия камеры корректно компенсируется на примерах снимков.
- Эпиполярные линии показывают геометрическую связь между кадрами, найденные inliers демонстрируют качество сопоставления точек.
- Для улучшения результатов можно увеличить количество кадров шахматки и их перекрытие, а также использовать более информативные сцены для ORB.
