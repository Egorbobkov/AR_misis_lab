# 3D-реконструкция объекта с помощью COLMAP

## Цель работы
Получить 3D-реконструкцию объекта по набору фотографий и на практике разобраться
в этапах фотограмметрического пайплайна: **feature extraction → matching → sparse → dense reconstruction**.
Оценить влияние качества съёмки на итоговую модель.

---

## Исходные данные
- Камера: смартфон
- Количество изображений: 29
- Формат: JPEG
- Съёмка выполнена по кругу вокруг объекта
- Объект имеет выраженную текстуру, без сильных бликов и прозрачных элементов

---
## Выводы о качестве съёмки и реконструкции

По результатам реконструкции COLMAP смог корректно использовать лишь часть исходных фотографий, что привело к неполной и фрагментированной 3D-модели. Это указывает на то, что условия съёмки оказались не оптимальными для фотограмметрической реконструкции.

### Основные факторы, повлиявшие на качество результата

**Недостаточная связность набора изображений**  
Несмотря на общее количество кадров, между многими фотографиями оказалось слишком мало совпадающих ключевых точек. В результате COLMAP не смог объединить все изображения в одну связную модель.

**Особенности выбранной стратегии съёмки**  
Съёмка выполнялась с двух уровней по высоте и углу обзора. Вероятно, переход между этими уровнями был слишком резким, из-за чего алгоритм не смог надёжно сопоставить признаки между группами изображений.

**Ограниченное количество информативных деталей на объекте**  
Часть изображений содержит области с низкой текстурированностью или бликами, что негативно сказалось на качестве детекции и сопоставления ключевых точек.

**Отсутствие настройки параметров реконструкции**  
Все этапы (Feature Extraction, Matching, Mapping) выполнялись с параметрами по умолчанию, без подбора под конкретную сцену, что могло привести к отбрасыванию потенциально полезных изображений.

### Итоговая оценка

Полученная 3D-модель отражает лишь часть геометрии объекта и не может считаться качественной реконструкцией. Тем не менее, эксперимент показал корректную работу пайплайна и выявил ключевые ограничения, связанные с качеством исходных данных.

### Возможные улучшения

Для улучшения результата в дальнейшем требуется:
- повысить перекрытие между кадрами и обеспечить более плавное изменение ракурсов;
- уменьшить различия между уровнями съёмки;
- при необходимости адаптировать параметры COLMAP под конкретную сцену.
